Detailed Description

template<std::size_t I>
struct TensorIndex< I >

Represents the indices in a TensorExpression.

Details

Used to denote a tensor index in a tensor slot. This allows the following type of expressions to work:

auto T = evaluate<ti_a_t, ti_b_t>(F(ti_a, ti_b) + S(ti_b, ti_a));

where using ti_a_t = TensorIndex<0>; and TensorIndex<0> ti_a;, that is, ti_a and ti_b are place holders for objects of type TensorIndex<0> and TensorIndex<1> respectively.
