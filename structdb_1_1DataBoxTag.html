Detailed Description

Tags for the DataBox inherit from this type.

Details

Used to mark a type as being a DataBoxTag so that it can be used in a DataBox.

Derived Class Requires:

  • type alias type of the type this DataBoxTag represents
  • static constexpr DataBoxString_t that is the same as the type name and named label

Example

struct Tag0 : db::DataBoxTag {
using type = double;
static constexpr db::DataBoxString_t label = "Tag0";
};
